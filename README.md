# MERNATAG Projects

MERNATAG = MERN + ATAG

Consisting of the following tech stack:

- M - MongoDB
- E - Express
- R - React
- N - Node
- A - Ant Design
- T - TypeScript
- A - Apollo
- G - GraphQL

All projects are split into server and client apps.
